## ⚡ The Vision

This isn't just a static resume. It's a high-performance, interactive playground built to showcase the intersection of **Enterprise AI Engineering** and **Cutting-edge Web Aesthetics**. From an interactive terminal that simulates an AI server to retro simulations hidden in the latent space, every pixel is engineered to engage.

## 🛠️ Technology Stack

| Layer | Technologies |
| :--- | :--- |
| **Core** | Semantic HTML5, Vanilla JavaScript (ES6+), Modern CSS (Variables & Grid) |
| **Special Features** | Interactive Terminal, Pacman Simulation|
| **Animations** | Glitch.js (custom), Tilt.js logic, Intersection Observer API, CSS Keyframes |
| **SEO & Performance** | Schema.org JSON-LD, OpenGraph, Twitter Cards, AVIF Compression |
| **Infrastructure** | Vercel Deployment, Speed Insights, UptimeRobot (Multi-Region Monitoring) |

## 🚀 Key Features

-   **🕹️ Geek Zone Terminal**: A fully functional interactive shell. Type `help` to explore projects, skills, or launch the `pacman` simulation.
-   **🌓 Dynamic Theme Engine**: Seamless switching between Dark and Light modes with persistent state and coordinated asset transitions.
-   **✨ Fluid Interactivity**: Custom pointer-glow effects, 3D tilt cards, and scroll-aware side navigation for a premium "app-like" feel.
-   **🕵️ Hidden Easter Eggs**: Type `anitej` or type `ai` anywhere on the site to trigger advanced visual overrides and system shifts.
-   **🤖 AI-Themed 404**: A "Latent Space Not Found" experience that keeps the user immersed even when they lose their way.
-   **📈 System Monitoring**: A dedicated [/status](https://anitej.online/status) page tracking uptime across Asia and US regions.

## 🛠️ Quick Start

No heavy dependencies or build steps. Pure, optimized vanilla code.

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/AnitejSood/portfolio.git
    cd portfolio
    ```

2.  **Run Locally**
    You can simply open `index.html` in your browser, or use a simple server for the full experience (including status routing):
    ```bash
    npx http-server .
    ```
